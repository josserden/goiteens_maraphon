Tech Skills

Щомісяця ви будете вивчати одну мову програмування і робити свій проект

  • JavaScript

    У перший місяць ви зробите свій власний сайт-календар цілей, в якому за допомогою HTML/CSS зможете візуалізувати свої 3 мети, продумає якими навичками для цього потрібно володіти та за допомогою JavaScript зробите динамічні графіки. Створений календар можна буде роздрукувати і повісити собі на стіну.

  • Java

    На другий місяць ви зробите Телеграм-Бот на Java, який буде рахувати скільки місяців вам знадобиться щоб заробити на мрію, працюючи в тій чи іншій професії. Вивчіть основи Java Core, навчитеся працювати з алгоритмами, обробляти тексти.

  • Python

    Ви навчитеся парсити дані, обробляти інформацію і робити прогнози курсів криптовалюти. При цьому, результат роботи можна буде передати вже створеному чат боту на Java і візуалізувати на JavaScript.
